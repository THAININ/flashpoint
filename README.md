# Projeto para estudar stack DevOps.

### 🛠️ Tecnologias
* **Python / Django 6.0**: Framework backend.
* **Docker & Docker Compose**: Containerização e orquestração.
* **Terraform**: Gerente de infraestrutura que cria e organiza automaticamente os serviços na nuvem (S3, Redes, Bancos de Dados) via código.
* **LocalStack**: Emulação de serviços AWS (S3).
* **GitHub Actions**: Pipeline de CI/CD.
* **AWS CLI**: Interface de linha de comando para validação.
